# Light Flyer

Local Environment for createing Lapps.

## Usage

```
npm install

npm link
```


## Commands

```
# Help & Commands
lightflyer -h

# Version
lightflyer -v

# Basic Commands
lightflyer start 
lightflyer stop

# Oparetate Own Node
lightflyer 


# Oparetate Other Node
lightflyer peer alice bob

```
